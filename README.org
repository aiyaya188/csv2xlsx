* csv2xlsx
** Introduction
Because [[https://github.com/tealeg/xlsx2csv][the XLSX2CSV program]] exists, it seemed that it was natural to
create a csv2xlsx corollary when the underlying xlsx library grew the
ability to write XLSX files.

** Alternatives
